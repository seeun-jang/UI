
# PLC 자동운전 GUI 프로그램  

### 📌 과제 개요

PLC 시뮬레이터와 C# Windows Forms GUI 프로그램을 연결하여 자동운전 공정을 구현한 과제이다.
GUI의 연결, 시작, 정지 버튼을 이용해 시뮬레이터를 제어하고, 리프트센서에 물건이 감지되면 B실린더와 C실린더가 자동으로 전진 및 후진하도록 구성하였다.

### ⚙️ 주요 기능  
- PLC 시뮬레이터 연결 기능  
- 자동운전 시작 및 정지 기능  
- 리프트센서 A, B 감지
- B실린더 전진/후진 자동 제어
- C실린더 전진/후진 자동 제어
- 센서 조건에 따른 단계별 자동운전 처리

### 🔁 자동운전 동작 순서
1. 리프트센서 A에 물건 감지
2. B실린더 전진
3. B실린더 전진 완료 후 후진
4. 리프트센서 B에 물건 감지
5. C실린더 전진
6. C실린더 전진 완료 후 후진
7. 다음 물건 대기
